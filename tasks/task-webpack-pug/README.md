# `@io-arc/task-webpack-pug`

Pug compile for webpack.

See the [documents](https://io-arc.tech/plugins/task-webpack-pug.html).

## Install

```shell
$ npm i -D @io-arc/task-webpack-pug
```

## Usage

```typescript
// webpack.config.js
const html = require('@io-arc/task-webpack-pug').html

module.exports = html
```
