# `@io-arc/task-webpack-stylus`

Stylus compile for webpack.

See the [documents](https://io-arc.tech/plugins/tasks/webpack-stylus.html).

## Install

```shell
$ npm i -D @io-arc/task-webpack-stylus
```

## Usage

```javascript
// webpack.config.js
const { css } = require('@io-arc/task-webpack-stylus')

module.exports = css
```
