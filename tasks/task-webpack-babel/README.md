# `@io-arc/task-webpack-babel`

Babel compile for webpack.

See the [documents](https://io-arc.tech/plugins/tasks/webpack-babel.html).

## Install

```shell
$ npm i -D @io-arc/task-webpack-babel
```

## Usage

```javascript
const { js } = require('@io-arc/task-webpack-babel')

module.exports = js
```
