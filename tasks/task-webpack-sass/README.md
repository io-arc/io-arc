# `@io-arc/task-webpack-sass`

Sass (SCSS/SASS) compile for webpack.

See the [documents](https://io-arc.tech/plugins/task-webpack-sass.html).

## Install

```shell
$ npm i -D @io-arc/task-webpack-sass
```

## Usage

```javascript
// webpack.config.js
const { css } = require('@io-arc/task-webpack-sass')

module.exports = css
```
